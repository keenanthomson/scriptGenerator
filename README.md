# scriptGenerator

App for generating commonly-used SQL scripts for A/B test analysis.

Preview live app here (https://scriptgenerator.glitch.me/).

## Getting Started

### Prerequisites
In your terminal,
- [ ] `npm install`

### Starting the App
In your terminal,
- [ ] Run webpack `npm run build-dev`
- [ ] Start your server `npm start`
